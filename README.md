# sams_helper

山姆下单助手

修改自： https://github.com/robGoods/sams

## 使用方式

```go
err := session.InitSession("<auth-token>", 2)
```

### 参数说明

1. 第一个参数：Sam's 登录后的 Header `auth-token`
2. 第二个参数：1：急速达，2：全城配送，目前上海地区只开放了全城配送故默认为 2
